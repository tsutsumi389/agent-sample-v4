このエージェントの特徴

🏠

ローカルで完結

Ollama 上の LLM を使うため、外部APIに会話を送らずローカルで動作します。

🔌

コア無改修で拡張

ツールはファイルを置くだけ、MCPはJSONに1行足すだけ。graph.py は触りません。

🧩

使うほど最適化

LangMem がバックグラウンドで会話から記憶を抽出・統合し、あなたに合わせます。

🛡️

LLM性能に依存しない堅牢さ

ルーティング・停止判定はコードが決定論的に制御。全経路が必ず終了します。
